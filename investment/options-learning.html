期权价值

公式：

Call: c = SN(d₁) - Ke-rTN(d₂)
Put: p = Ke-rTN(-d₂) - SN(-d₁)
d₁: d₁ = [ln(S/K) + (r + σ2/2)T] / (σ√T)
d₂: d₂ = [ln(S/K) + (r - σ2/2)T] / (σ√T) = d₁ - σ√T

波动率

期权价格，受 股价S，行权价K，到期时间T，无风险率R，以及波动率σ Volatility 影响。 波动率，最难估算。其定义是：股价单日回报率的标准差。 这个很难计算，即是计算出来，也很难跟上市场的变化。 一般是通过期权价格，反推波动率。这个波动率，就是隐含波动率（Implied Volatility, IV）。 我们通过IV，判断期权价格是否合理。 这个和用估值PE 或者 PS判断股票是否合理一样，IV判断期权价格是否合理。 一般参考当前IV，在52 Wk IV的分位数，来判断期权价格是否合理，一般20%以内，则认为合理。 也可以把当前的期权，和其他股票的期权进行对比。

期权价格曲线

  • call option
    • 到期收益曲线，和横坐标的距离，就是内在价值。（期权到期，只有内在，没有时间）

    到期收益曲线图

    可以看的出来，股价越高，期权价格对股价约敏感，反之则不敏感。 所以，当做带方向期权的时候，敏感度越高越好；做波动率期权的时候，敏感度越低越好。当然，这里指的是做多。

流动性

判断期权流动性

  • bid-ask spread
  • 行权价选择越多，流动性越好

    • 不仅看绝对值，还要看相对值

流动性规则：
  • 每月第三个星期五，即是月期权，流动性更好
  • ATM期权，流动性最好
  • OTM 比 ITM 流动性更好
  • 行权价，是整数的，流动性更好。即0 5 结尾
  • 到期时间，越接近到期，流动性越好
